API Examples

Common use cases and code examples

Create Employee

Add a new employee to your organization

const response = await fetch('https://api.talentoze.com/v2/employees', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    firstName: 'John',
    lastName: 'Doe',
    email: 'john.doe@company.com',
    department: 'Engineering',
    position: 'Software Engineer',
    startDate: '2024-07-25'
  })
});

const employee = await response.json();
console.log('Employee created:', employee);

Process Payroll

Run payroll for a specific pay period

import requests

url = "https://api.talentoze.com/v2/payroll/process"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "payPeriodStart": "2024-07-01",
    "payPeriodEnd": "2024-07-15",
    "employees": ["emp_123", "emp_456"],
    "dryRun": False
}

response = requests.post(url, headers=headers, json=data)
payroll_result = response.json()
print(f"Payroll processed: {payroll_result['status']}")

curl -X GET \

Generate Report

Create custom reports with filters

const reportData = {
  type: 'headcount',
  filters: {
    department: ['Engineering', 'Marketing'],
    dateRange: {
      start: '2024-01-01',
      end: '2024-07-25'
    }
  },
  format: 'json'
};

fetch('https://api.talentoze.com/v2/reports/generate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(reportData)
})
.then(response => response.json())
.then(report => console.log('Report generated:', report));

Authentication

Secure your API requests with proper authentication

API Key Authentication

Include your API key in the Authorization header of every request:

Authorization: Bearer YOUR_API_KEY

OAuth 2.0

For applications requiring user-specific access, use OAuth 2.0 flow:

// Step 1: Redirect user to authorization URL
const authUrl = 'https://api.talentoze.com/oauth/authorize?' +
  'client_id=YOUR_CLIENT_ID&' +
  'redirect_uri=YOUR_REDIRECT_URI&' +
  'response_type=code&' +
  'scope=employees:read payroll:write';

// Step 2: Exchange code for access token
const tokenResponse = await fetch('https://api.talentoze.com/oauth/token', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    grant_type: 'authorization_code',
    client_id: 'YOUR_CLIENT_ID',
    client_secret: 'YOUR_CLIENT_SECRET',
    code: 'AUTHORIZATION_CODE',
    redirect_uri: 'YOUR_REDIRECT_URI'
  })
});

Webhooks

Receive real-time notifications for important events

Available Events

  • employee.created - New employee added
  • employee.updated - Employee information changed
  • payroll.processed - Payroll run completed
  • performance.review_completed - Performance review submitted
  • time.clock_in - Employee clocked in
  • time.clock_out - Employee clocked out

Webhook Payload Example

  "event": "employee.created",

Official SDKs

Use our official SDKs for faster development

JavaScript SDK

Full-featured SDK for Node.js and browser environments

npm install @talentoze/sdk
import Talentoze from '@talentoze/sdk';

const client = new Talentoze({
  apiKey: 'YOUR_API_KEY'
});

const employees = await client.employees.list();
console.log(employees);

Python SDK

Pythonic interface for all Talentoze API endpoints

pip install talentoze-python
from talentoze import Client

client = Client(api_key='YOUR_API_KEY')

employees = client.employees.list()
for employee in employees:
    print(f"{employee.first_name} {employee.last_name}")
